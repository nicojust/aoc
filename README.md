## 2024 [python]

### [Virtual Environment](https://python-poetry.org/docs/basic-usage/#activating-the-virtual-environment)

* Activate Environment

    ```sh
    make activate
    ```

* Deactivate Environment

    ```sh
    make deactivate
    ```

### Run

```sh
poetry run python src/Year2024/aoc.py 1 2024 prod
```
